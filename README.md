# ScrimsTracker

A school project focused on designing a homepage for a scrim tracking platform aimed at competitive gamers.

## About

ScrimsTracker is a concept web platform where competitive players can track their scrim results, analyze performance stats, and climb leaderboards. This project was developed as part of a school assignment, covering UI/UX design principles and visual identity.

## What's Included

- Homepage design (dark mode & light mode)
- Navigation bar with links to Home, Games, Scrims, Leaderboard, and Profile
- Hero section with call-to-action buttons
- Stats overview with charts and performance data
- Leaderboard preview
- Responsive and clean layout

## Tools Used

- Figma — UI/UX design
- UX Pilot — AI-assisted design generation

## Status

This is a school project and is not intended for production use.
